# YouTube Clone

A responsive front-end clone of YouTube built using **HTML5** and **CSS3**. This project replicates the basic user interface of YouTube, including the navigation header, sidebar, search functionality, video recommendation grid, channel information, and responsive layout design.

## Features

* YouTube-inspired user interface
* Responsive header with:

  * Hamburger menu
  * YouTube logo
  * Search bar
  * Voice search button
  * Notifications
  * User profile icon
* Sidebar navigation

  * Home
  * Explore
  * Shorts
  * Subscriptions
  * Settings
* Video recommendation grid
* Video thumbnails with duration labels
* Channel profile pictures
* Video metadata display

  * Title
  * Channel name
  * Views
  * Upload time
* Clean and organized CSS structure

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/youtube-clone.git
```

2. Navigate to the project folder:

```bash
cd youtube-clone
```

3. Open `youtube.html` in your browser.
